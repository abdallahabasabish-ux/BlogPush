# 🚀 BlogPush – منصة الإشعارات الفورية للمدونين (Blogger)

منصة متكاملة تسمح لأصحاب مدونات Blogger بجمع المشتركين وإرسال إشعارات فورية وإدارة الحملات بكل سهولة.

## 🛠️ التقنيات المستخدمة

- **الواجهة الأمامية**: Blogger + HTML5 + CSS3 + Vanilla JavaScript + Bootstrap 5
- **الخادم الخلفي**: Google Apps Script (REST API)
- **قاعدة البيانات**: Google Sheets
- **التخزين**: Google Drive
- **الإشعارات**: Firebase Cloud Messaging (FCM)
